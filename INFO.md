## Prerequisites

1. `npm install -g npm@8.13.2`
2. [node-v18.5.0-x64.msi](https://nodejs.org/dist/v18.5.0/node-v18.5.0-x64.msi)
3. Nuxt CLI v3.0.0-rc.4

-----------------

## Nuxt project create

```shell
npx nuxi init .
npx nuxi init nuxt-client-app

npm i
npm audit fix 
npm run dev
```

## Start Nuxt app in development mode:

```shell
npm run dev -- -o
```
